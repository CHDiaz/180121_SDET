DIRECTIVES

Directives are ways that we can invoke Angular functionality within an html file. They act as syntax that can be used in html format without breaking the front end.

*ngIf

What does the asterisk(*) mean in front of the ngIf?

Any directive that has an asterisk(*) means that it is a structural directive. This means the directive is capable of changing the actual structure of the HTML itself. IE, it can add/remove elements on the page.

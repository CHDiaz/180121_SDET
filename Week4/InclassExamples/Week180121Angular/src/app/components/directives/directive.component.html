DIRECTIVES

Directives are ways taht we can invoke Angular functionality within an html file. They act as syntax that can be used in html format without breaking the front end.

*ngIf

What does the asterisk(*) mean in front of the ngIf?

Any directive that has an asterisk(*) means that is is a structural directive. This means the directive is capable of changing the actual structure of the HTML itself. IE, it can add/remove elements on the page.

*ngFor

*ngFor is a directive that can add elements based off of some count of items from a array. For each item we can create a duplicate representation of the tag and children tags we apply it to. With the data being different thanks to data binding. (Since this is changing the structure of out HTML page, we can call this a structural directive.)

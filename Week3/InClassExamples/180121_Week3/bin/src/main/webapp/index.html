HTML (Hypertext Mark-up Language)

Everything in HTML is done using tags. This is how a markup language is formatted. Different tags perform different actions. The above header is done using the <h1> tag.

Any tag can be called an element. An key value pairs within the element are called attributes.

Hypertext Markup languages are used to help a browser render content on screen. There is no compilation, it is all interpreted. (Means that the code, as is, is used to display content)

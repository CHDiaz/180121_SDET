Can you explain syntactically, what is going on here? - Jawaun

With radio button types, you can have multiple radio buttons that may be toggled. However, in order to enforce that only one of a specific group can be toggled, you need to ensure they each have the same name. Since the rule goes: Only one radio button of the same group can be selected at a time. The value attribute is used to store which one is toggled.
